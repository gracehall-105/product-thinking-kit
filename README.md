# Product Thinking Kit
A toolkit that helps builders evaluate whether their product idea has a real market opportunity, a clear target user, a right to win, and messaging that makes the value obvious.
This repository contains prompts designed to help builders pressure-test product ideas and refine positioning before building more features.
The prompts are designed to work with AI development tools to analyze an app's strategy, positioning, and discoverability.

## Product Judgment Flow

This toolkit helps builders move through a structured product judgment process.

Market  
↓  
Problem  
↓  
Right to Win  
↓  
Target User  
↓  
Wedge  
↓  
Messaging  
↓  
Findability  
↓  
Ship

The prompts in this repository guide builders through each step.
